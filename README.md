# Student Records DataFrame Project

## Overview
This project demonstrates how to create and manipulate a Pandas DataFrame containing student records. It showcases different data types and uses `select_dtypes()` to filter columns based on their data types.

## Features
- Create a DataFrame from a Python dictionary
- Display complete student records
- View column data types
- Extract numeric columns
- Extract string columns
- Extract boolean columns
- Extract non-numeric columns

## Technologies Used
- Python
- Pandas

## Dataset Fields
| Column Name | Description |
|------------|-------------|
| RollNo | Student Roll Number |
| StudentName | Student Name |
| Department | Department Name |
| CGPA | Student CGPA |
| Attendance | Attendance Percentage |
| Age | Student Age |
| Hosteller | Hostel Status (True/False) |

## How to Run

1. Install Pandas:
   ```bash
   pip install pandas
   ```

2. Run the Python file:
   ```bash
   python student_records.py
   ```
